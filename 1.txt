УП.01
Разработка аналога системных утилит GNU Linux на языке Golang

Общее задание:

Используя интегрированную среду разработки IDE требуется разработать аналоги системных утилит в соответствии с заданием на соответствующую оценку в ОС GNU Linux
Для разработки допускается использовать только язык программирования Golang. Использовать любой другой язык программирования запрещено.
Реализацию каждой команды необходимо выполнить в отдельном исполняемом файле.
В коде реализации каждой команды проекта обязательно должны быть реализованы комментарии. Реализовать использование различных структур данных из языка Golang
Для каждой утилиты необходимо реализовать свой набор входных аргументов в количестве 3 штук. Аргумент -h необходимо реализовать в каждой команде, он является обязательным.
Использовать любые сторонние внешние подключаемые библиотеки запрещено.
Использовать функцию exec из состава библиотеки языка Golang для вызова аналогичной системной утилиты запрещено
Обязательно предусмотреть контроль ввода начальных/входных значений на корректность/правильность (например, выход за диапозон указанных чисел/значений) используя механизм обработки исключительных ситуаций! Если при сдаче проекта приложение/команда внезапно "упадет" или "зависнет", из общей оценки вычитается ровно 1 балл!

Задание на оценку 5
Команды ls -R -a -l -r -h, cd -, mkdir -p, rmdir, cat -n, file, nl, zip, unzip, tar -c -f -x, !!, !n, free, rm -R -f, head -c -n, date, arch, ps, du, df, wc, kill, tail -n, uname, cp, history, pwgen, hexdump, find
